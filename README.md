Laze - OpenGL Edition
=====================

О ПРОЕКТЕ
---------
Laze - это атмосферная 3D-игра в жанре лабиринт с ретро-эстетикой в стиле Synthwave. 
Вы оказываетесь в бесконечно генерируемых лабиринтах под высоким звездным небом и неоновым солнцем.
Ваша цель - найти выход из лабиринта, чтобы перейти на следующий, более сложный уровень.

ОСОБЕННОСТИ
-----------
* Процедурная генерация уровней: каждый лабиринт уникален.
* Ретро-графика: Synthwave стиль, объемное освещение и звездное небо.
* Живое меню: игра продолжается на фоне, даже когда вы находитесь в меню.
* Система настроек: регулировка громкости и чувствительности мыши.
* Поддержка своей музыки: игра воспроизводит треки из папки 'music'.

УПРАВЛЕНИЕ
----------
[W, A, S, D] - Перемещение
[Мышь]       - Оглядеться
[Shift]      - Бег
[Space]      - Прыжок
[Tab]        - Показать мини-карту (удерживать)
[Esc]        - Пауза / Назад

В МЕНЮ
------
[Стрелки]    - Навигация по пунктам
[Enter]- Выбор пункта
[Esc]        - Назад / Вернуться в игру

УСТАНОВКА МУЗЫКИ
----------------
Чтобы в игре играла музыка, поместите файлы формата .mp3, .ogg или .wav в папку 'music',
находящуюся рядом с исполняемым файлом игры.

Приятной игры!
